TCViewer Support

TCViewer 是一个用于在 iPhone 上浏览兼容 TeslaCam 目录结构视频的工具。将外部存储设备连接到 iPhone 后，你可以直接选择 TeslaCam 文件夹并回看行车记录。

TCViewer is an iPhone app for browsing videos stored in a TeslaCam-compatible folder. After connecting an external drive to your iPhone, you can select the TeslaCam folder and review dashcam footage directly on your phone.

Last updated: 2026-03-30

如何使用

  1. 将包含 TeslaCam 目录的外部存储设备连接到 iPhone。
  2. 打开 TCViewer，点击“选目录”。
  3. 选择 TeslaCam 文件夹，或选择包含 TeslaCam 的上级目录。
  4. 等待扫描完成后，在事件列表中浏览并播放视频。
  1. Connect an external storage device that contains a TeslaCam folder to your iPhone.
  2. Open TCViewer and tap "Select Folder".
  3. Choose the TeslaCam folder, or its parent folder if TeslaCam is inside it.
  4. Wait for the scan to finish, then browse events and play videos.

常见问题

  • 如果提示未找到 TeslaCam 目录，请确认你选择的是 TeslaCam 文件夹本身，或者它的直接上级目录。
  • 如果没有显示事件，请确认外部存储中包含兼容的 `.mp4`、`.mov` 或 `.m4v` 视频文件。
  • 如果缩略图或播放异常，请重新扫描一次，或重新插拔外部存储设备后重试。
  • 如果问题仍然存在，请发送邮件联系开发者，并尽量附上目录结构说明、iPhone 型号和系统版本。
  • If the app says it cannot find the TeslaCam folder, make sure you selected the TeslaCam folder itself or its direct parent folder.
  • If no events are shown, confirm that the external drive contains compatible `.mp4`, `.mov`, or `.m4v` video files.
  • If thumbnails or playback behave unexpectedly, rescan the folder or reconnect the external drive and try again.
  • If the issue persists, contact the developer by email and include the folder structure, your iPhone model, and iOS version when possible.

联系与反馈

目前支持渠道为电子邮箱。来信时请尽量附上重现步骤、机型、iOS 版本以及 TeslaCam 目录情况，这会帮助更快定位问题。

Support is provided by email. When possible, include reproduction steps, your device model, iOS version, and the TeslaCam folder structure so the issue can be investigated more quickly.
