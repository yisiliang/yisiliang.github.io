TCViewer 隐私政策

Last updated: 2026-03-30

1. 概述

TCViewer 是一个用于在 iPhone 上查看兼容 TeslaCam 目录结构视频内容的工具。我们重视你的隐私，并尽可能减少数据收集与传输。

2. 我们收集哪些信息

TCViewer 不会创建账号，也不会主动收集以下信息：

  • 姓名、邮箱、电话等身份信息
  • 位置信息
  • 联系人
  • 照片图库内容
  • 使用分析数据
  • 广告标识符或追踪数据

3. App 如何处理你的文件

当你在 App 中手动选择目录后，TCViewer 会在设备本地访问你授权的目录，用于：

  • 扫描 TeslaCam 目录中的视频文件
  • 读取必要的文件名、文件时间和视频时长等信息
  • 生成本地缩略图
  • 按事件组织并播放视频

这些操作都在你的设备本地完成，不会将视频文件上传到开发者服务器。

4. 本地存储内容

为提升使用体验，TCViewer 可能会在设备本地保存以下内容：

  • 你授权过的目录书签
  • 已扫描的事件索引缓存
  • 视频缩略图缓存

这些内容仅用于 App 本地功能，不会被出售或共享给第三方。

5. 第三方服务

截至本隐私政策更新日期，TCViewer 未集成第三方广告、统计或用户跟踪 SDK。

6. 数据共享

TCViewer 不会将你的 TeslaCam 视频内容、缓存索引或缩略图出售、出租或共享给第三方。

7. 儿童隐私

TCViewer 不面向儿童提供社交或广告功能，也不会有意收集儿童个人信息。

8. 你的控制权

  • 取消选择目录
  • 删除 App，从设备上移除缓存内容
  • 停止继续向 App 授予外部目录访问权限

9. 联系方式

如有隐私相关问题，请通过 yisiliang@foxmail.com 或 GitHub Issues 联系开发者。

10. 政策更新

如果 TCViewer 的隐私实践发生变化，我们会在发布新版本时更新本隐私政策。
